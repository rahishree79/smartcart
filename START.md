# How to Run SmartCart

## Step 1: Install dependencies

```bash
# Terminal 1 – Backend
cd backend
npm install
npm run seed    # Seeds database with 12 products + test user
npm run dev     # Runs on http://localhost:5000

# Terminal 2 – Frontend
cd frontend
npm install
npm run dev     # Runs on http://localhost:5173
```

## Step 2: Open browser
Go to: http://localhost:5173
